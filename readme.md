Serdivan is a attribute based simple grid system.
Visit [Homepage](http://ahmetozantekin.github.io/serdivan/).

## Install

#### [npm]()
```
$ npm install serdivan
```

#### [bower]()
```
$ bower install serdivan
```

#### [yarn]()
```
$ yarn add serdivan
```
